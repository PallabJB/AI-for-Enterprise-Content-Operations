# 🚀 Enterprise Content Operations Swarm

> **Automate the full lifecycle of enterprise content: creation, compliance, localization, and multi-channel distribution.**

![Enterprise AI Dashboard Preview](https://img.shields.io/badge/Status-Hackathon--Ready-brightgreen)
![Tech Stack](https://img.shields.io/badge/Tech-FastAPI%20%7C%20React%20%7C%20Vite%20%7C%20Tailwind-blue)

## 🌟 The Problem
Scaling content in a global enterprise is a **human bottleneck**. 
- Taking a technical report to a social post takes **72+ hours**.
- Manual legal/compliance reviews are slow and error-prone.
- Localization into 10+ languages stalls global campaigns.

## 🧠 Our Solution: The Agentic Swarm
**Antigravity AI** deploys a collaborative swarm of specialized agents that handle the heavy lifting of content operations **autonomously**. 

- **Knowledge Analyst Agent**: Extracts facts from raw internal data (Reports, Specs, Meeting Notes).
- **Global Copywriter Agent**: Drafts platform-specific (LinkedIn, Twitter, CMS) assets in seconds.
- **Governance Guard Agent**: Performs real-time brand & legal audits (Compliance Score).
- **Localization Expert Agent**: Adapts content for global regions (ES, FR, DE) with cultural context.

---

## 🛠️ Tech Stack
- **Frontend**: Vite + React + TypeScript + Framer Motion + Tailwind CSS + Lucide Icons.
- **Backend**: Python + FastAPI + Pydantic + Uvicorn.
- **Orchestration**: Custom Agentic State Machine with background tasks.
- **Design Style**: Premium, modern dark-mode with glassmorphism and animated analytics.

## 🚀 Getting Started

### 1. Backend Setup
```bash
cd backend
pip install -r requirements.txt
python main.py
```
*Accessible at: `http://localhost:8000`*

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
*Accessible at: `http://localhost:3000`*

## ✨ Key Features
- **Swarm Operations Center**: Real-time visualization of agent progress and pipeline state.
- **Content Intelligence**: Animated analytics dashboard showing engagement, ROI, and cycle health.
- **Compliance Guardrails**: Automatic flagging of sensitive terms (e.g., claiming "zero risk") with a safety score.
- **Full-Screen Optimization**: Designed for wide-screen operational desks in corporate centers.
